# HAPPY TO DO

'Happy To do it'의 뜻은 '기꺼이 맡아서 할게'이다.
하루동안의 To do list에 적힌 일을 맡아서 수행하면서 성장하고 싶은 개발자이고
'서희'라는 이름의 기쁠 희 한자를 가져와 'Happy'라는 단어를 선택하여 만들어진 이름이다.
